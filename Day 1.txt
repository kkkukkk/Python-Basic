파이썬이란??
파이썬은 1991년 네덜란드의 수학자 '귀도 반 로섬'에 의해 개발된 프로그래밍 언어
'비단뱀'에서 유래한 로고를 볼 수 있다.

구글의 많은 앱들과 인스타그램 등의 실용적인 앱들도 파이썬으로 만들어졌다.

1) 인터프리터 언어이다. (인터프리터는 한 줄씩 실행, line by line)
<=> (컴파일러 언어는 덩어리로 실행)

2) 플랫폼에 독립적이다. (어떠한 운영체제에서도 같은 파이썬 코드를 실행할 수 있다, windows, osx(MAC), linux 등)

3) 객체 지향언어이다. (object oriented 클래스와 객체를 기반으로 하는 언어로 정보은닉, 상속 등의 개념을 지원한다.)

4) 동적 타이핑 dynamic typing 언어이다. (프로그램 실행 중에 다른 작업을 처리할 수 있다.)

5) 개발 속도가 빠르다. (이미 완성된 많은 다양한 라이브러리)
-> (데이터 분석 pandas, 수치연산 numpy, 시각화 matplotlib, Django, flask 웹개발, science kit learning 딥러닝 등등을 사용하여 빠르게 프로그램을 작성할 수 있다.)

* 하드웨어, 앱개발(아이폰, 안드로이드) 등에는 적용하기 어렵다.

##############
# 주석 comment
##############

주석은 프로그램 소스 코드에 설명을 작성한 것을 의미한다.
주석은 프로그램 실행에 전혀 영향을 끼치지 않는다. 
유지 보수나 팀작업을 할 때 모두가 이해하기 쉽고 의미 전달이 잘 되는 주석을 작성하는 것이 바람직하다.
파이썬에서 주석은 #으로 시작하거나 ''' ''' 삼중 따옴표로 표현한다.

변수는 어떤 데이터를 저장할 때 사용하는 메모리 저장소(?)이다.
A-
변수를 만들때는 = 등호 왼쪽에 변수 명을 작성하고 등호 오른쪽에는 저장할 값을 작성한다.

변수명 규칙
1. 영문, 한글, 숫자, 밑줄로 구성된다. (변수명으로 한글은 쓰지 말자)
2. 특수문자는 사용할 수 없다.
(! @ # $ % ......)
3. 대문자와 소문자를 구분한다.
4. 변수명의 첫글자는 숫자를 사용할 수 없다.
5. 예약어(print, list, dict, if, for......)는 변수명으로 사용할 수 없다.

*. 가급적이면 소문자만 사용한다.
*. 한글은 사용하지 말자.
*. 의미있는 변수명을 사용하자.
